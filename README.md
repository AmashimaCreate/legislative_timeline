# 立法タイムライン（Legislative Timeline）

国会で**成立した法律**を、法案ごとに「やさしい要約 → 何が変わる → 争点 → 賛否の数 → 議員ごとの投票 → 成立までのプロセス」で見える化する静的サイトです。
[政治見える化プロジェクト](https://seiji-mieru.com/) の一部です。

## 収録範囲
- **第217〜221回国会で成立した法律 135本**（審議未了・廃案・継続審議中は含みません）。
- 各法案＝やさしい要約／何が変わる／（討論があれば）争点と両論の要旨／賛否の数／（参院記名なら）議員ごとの賛否／提出〜成立〜公布のプロセス図。

## データの出典（一次資料）
- **参議院** 本会議投票結果（議員別の賛否）
- **衆議院** 会派ごとの賛否・進行日付 … スマートニュース メディア研究所「議案データベース」（MIT License）を利用
- **国会会議録検索システム**（委員長報告・本会議討論）
- **e-Gov法令検索**（成立した法律の条文）

## 事実とAIの区別
- **事実**（賛否の数・議員名・日付・成立プロセス）は上記の公式記録です。
- **やさしい要約・賛否の理由**は、国会会議録（委員長報告・討論）をもとに AI（Claude Haiku）が作成し、🤖 印で明示しています。会議録が未公開の直近法案は「準備中」と表示します。

## 中立性の方針
- 賛否は**数字で機械的に判断**します（全会一致の法案には「両論」を作りません）。
- 棄権・少数会派の反対を「造反」などと色付けしません。
- すべての情報に一次資料へのリンクを添えています。

## 構成
- 完全な静的サイトです（`index.html` ＋ `data/bills.json`、CSS/JS はHTMLにインライン）。サーバ不要。
- ローカルで開く場合は簡易サーバ経由で：`python3 -m http.server` を起動し `http://localhost:8000/` を開いてください（`fetch` のため）。
- データを生成するスクリプト（取得・要約生成）は現在非公開です。サイトが安定したのち、別途オープンソース公開を検討します。
